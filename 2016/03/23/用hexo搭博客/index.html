  • 安装Node（hexo基于Node.js）

命令行：

brew install node

检验是否安装完毕：

node -v
npm -v

npm 是node.js 环境下的包管理器，npm官方站点速度很慢，因此需要更换源，可以使用以下3种方法：

(1)通过 config 配置指向国内镜像源

npm config set registry http://registry.cnpmjs.org //配置指向源

npm info express //下载安装第三方包

(2)通过 npm 命令指定下载源

npm --registry http://registry.cnpmjs.org info express

(3)在配置文件 ~/.npmrc 文件写入源地址

vim ~/.npmrc //打开配置文件

registry=https://registry.cnpmjs.org //写入配置文件

推荐最后一种方法，永久改变源地址。

hexo实际安装地址位于/usr/local/bin/hexo软链于/usr/local/lib/node_modules/hexo-cli/bin/hexo，配置文件nmprc地址位于/usr/local/lib/node_modules
