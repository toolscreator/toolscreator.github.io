首次LFS，需要注意的事项

  • 在完成“Constructing a Temporary System”后，每次进入chroot环境前别忘了“Mounting and Populating /dev”和“Mounting Virtual Kernel File Systems”，然后chroot：“chroot “$LFS” /tools/bin/env -i \
    HOME=/root \
    TERM=”$TERM” \
    PS1=’\u:\w\$ ‘ \
    PATH=/bin:/usr/bin:/sbin:/usr/sbin:/tools/bin \
    /tools/bin/bash –login +h”。在/bin/bash构建完成后，最后一行命令需换成“/bin/bash –login +h”。在系统软件全部完成安装后，把命令行中关于/tools的全部去掉或替换成根目录。
  • 在“Installing Basic System Software”一章中最重要的一个思想是：在chroot环境下，一旦你完成一个系统软件的安装后，立即在当前环境使用它，而不是继续使用前一章构建的/tools下的工具。例如，在安装好bash后立即执行exec /bin/bash –login +h切换到新安装的shell环境中。其他软件不需要显示进行类似的操作，因为在chroot环境中，已经在PATH变量中把“/bin:/usr/bin:/sbin:/usr/sbin”设置在/tools的前面。
